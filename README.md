# vueqa
这是问答社区项目的前端部分(前后端分离)
前端框架是vue
使用axios向后端发送请求
## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Run your tests
```
npm run test
```

### BUG

> 每次导入element ui样式总是不对，需要自己调整
